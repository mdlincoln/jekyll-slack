<@U0CRECW1G> (with The Emory Center for Digital Scholarship) and I worked out a GeoServer workflow script. For those using GeoServer to deliver geospatial data online, the script runs select GDAL processing on raster images to optimize for displaying in Leaflet and other web mapping libraries. It then uploads the processed files to GeoServer, fills in selected fields based on your metadata files, and publishes a layer. You can access the script and documentation here: Many thanks to Eric Willoughby at Georgia State University for helping us fine-tune the GDAL commands. The script is an evolving project, and we welcome any feedback.
